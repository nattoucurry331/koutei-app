# コウテイ.app

建設業の現場担当者向け、ブラウザだけで動く工程表(ガントチャート)作成ツール。

Excel手作りから卒業して、ドラッグだけで工程表を作って PDF で共有できます。

## 特徴

- **シンプル**: 工種を追加してセルをドラッグするだけ
- **柔軟な表示**: 日 / 半日 / 週 の3単位を切替可能
- **A4横でPDF出力**: 1枚に収まらない場合も「週圧縮 / 月分割 / 横長」を自動提案
- **表紙ページ**: 元請提出向けのA4縦表紙(自社名・担当者名入り)を生成
- **データは手元に**: ブラウザの localStorage に自動保存。サーバーには一切送信しません
- **JSON共有**: 含める項目を選んでJSONで書き出し、別PCや協力会社と共有可能
- **日本の祝日対応**: 土日と祝日を背景色で自動区別

## 開発

```bash
npm install
npm run dev      # ローカル起動 (http://localhost:5173)
npm run build    # 本番ビルド (dist/ に出力)
```

## 技術スタック

- React 18 + TypeScript + Vite
- jsPDF + html2canvas (PDF出力)
- localStorage / Blob API (データ管理)
- 静的Webアプリ (サーバー不要)
